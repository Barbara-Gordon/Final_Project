What we did

The Data

Using Beautiful Soup, we web-scraped the Miller Center's collection of presidential speeches and the Commission on Presidential Debates collection of debate transcripts.
We cleaned and dumped the data into a Mongo database for exploration and testing.

Machine Learning

We trained and tested our data on SkLearn's Multinomial Naeve Bayes algorithm and were able to predict the decade of a text within 50%.
We also utilized Indico.io's text analysis API to extract political and emotional trends from the text.

Visualizations

We created a Flask API to display a bootstrap template of our findings. Our visualizations were created with amCharts, a d3 based Javascript library.
